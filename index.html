Documentação da API

Endpoints disponíveis para consumo.

Rotas Principais

POST /receber

Recebe os dados de telemetria do dispositivo (formato JSON) e salva localmente no servidor.

GET /relatorio

Gera e disponibiliza para download um relatório em PDF (simples) com base nos últimos dados de telemetria recebidos.

POST /orquestrador

Orquestra a geração de um relatório PDF complexo e customizado com base em uma estrutura JSON enviada no body.

Inputs da Rota

1. Query Parameter (URL)
nome_arquivo (string) - Obrigatório.

Define o nome do arquivo PDF a ser gerado.
Exemplo de URL: /orquestrador?nome_arquivo=meu_relatorio.pdf

2. Corpo da Requisição (JSON Body)

Define o conteúdo e a estrutura do relatório. Deve ser um JSON contendo a chave "elementos", que é um array de "comandos" a serem executados em ordem.

Detalhamento dos "Elementos" (JSON Body)

Cada objeto {} dentro do array "elementos" é uma instrução para o gerador de PDF. A ordem é fundamental.

  • Estrutura do Documento:
    • {"add_capa": {...}}: Adiciona uma página de rosto.
    • {"secao": "..."}: Cria um título de seção principal (ex: 1. Título).
    • {"subsecao": "..."}: Cria um título de subseção (ex: 1.1. Título).
    • {"nova_pagina": null}: Força uma quebra de página.
    • {"espaco": 15}: Adiciona um espaço vertical (em mm).
  • Conteúdo de Texto:
    • {"paragrafo": "..."}: Adiciona um parágrafo de texto normal.
    • {"lista_numerada": [...]}: Cria uma lista numerada.
    • {"lista_bullet": [...]}: Cria uma lista com marcadores.
    • {"bloco_codigo": "..."}: Adiciona um bloco de código formatado.
  • Conteúdo Visual e Dados:
    • {"legenda_tabela": "..."}: Define a legenda para a próxima tabela.
    • {"tabela_pandas": {...}}: Cria uma tabela a partir dos dados.
    • {"legenda_figura": "..."}: Define a legenda para a próxima imagem/gráfico.
    • {"imagem": {...}}: Insere uma imagem (a partir de um caminho no servidor).
    • {"grafico_pandas": {...}}: Gera e insere um gráfico dinamicamente.

Exemplo de JSON Body (Estrutura Resumida)

{
  "elementos": [
    {
      "add_capa": {
        "instituicao": "Minha Empresa",
        "autor": "Relatórios API",
        "titulo": "Relatório Técnico"
      }
    },
    { "secao": "Introdução" },
    { "paragrafo": "Texto do parágrafo aqui..." },
    { "subsecao": "Análise de Dados" },
    { "legenda_tabela": "Legenda da Tabela." },
    {
      "tabela_pandas": {
        "data": [
          {"Mês": "Outubro", "Vendas": 12000},
          {"Mês": "Novembro", "Vendas": 15000}
        ]
      }
    },
    { "legenda_figura": "Legenda da Figura." },
    {
      "grafico_pandas": {
        "data": [...],
        "x_col": "Mês",
        "y_col": "Vendas",
        "tipo_grafico": "bar"
      }
    },
    { "nova_pagina": null },
    { "secao": "Conclusão" },
    { "paragrafo": "Texto da conclusão." }
  ]
}
